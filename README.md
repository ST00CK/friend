# 시스템 아키텍처 개요
# 소개

Friend 애플리케이션은 계층화된 아키텍처 패턴을 따르며, 다양한 계층에서 관심사를 분리합니다:

    API 계층: Gin 웹 프레임워크를 사용하여 HTTP 요청 및 응답 처리
    비즈니스 계층: 서비스 컴포넌트를 통한 애플리케이션 로직 구현
    데이터 접근 계층: 저장소를 통한 데이터베이스 작업 관리

이러한 분리는 코드베이스의 유지보수성, 테스트 가능성 및 유연성을 향상시킵니다.
